Stibium

Stibium is a web application that provides an interface similar to a traditional desktop environment (it imitates KDE) but uses a cloud service to store your files. It can be seen as a desktop enviromment to manage your files on the cloud.

The tecnology

Stibium is powered by PHP, SVG, and Javascript, so it's all Free Open Source Software.
We use the Ubuntu Font (Ubuntu Fonts) because is a free font, and it's very nice.

Supported Clouds

Actually, we are working to support Ubuntu One, Dropbox, ownCloud, and Google Docs.

History

Stibium has been originally created by Luca Tringali, during september 2011, and it has been released with GPL2 licence officially october the 14th (it was the 15th KDE anniversary).
The name "Stibium" came from the 51st element of the periodic table (also called "antimony"). Why? Because the stibium is the element of alchemy, and it has a nice appering. And it sounds as a good name for a program.

Requirements for the client

To access Stibium as a client, all you need is a web browser supporting HTML5 and a Javascript. For example, one of these:

Requirements for the server

The server that hosts Stibium must support php and OAuth. It's recommended to have at least 2GB of free space on the server, so Stibium can create some temporary files.
Obiously, it's a lot better if the server uses SSL to encrypt connection (it's not a good idea to send important data through an unsecure connection).
There are also some other programs that should be installed on the server to let some things work:
  • ffmpeg: video editing
  • imagemagick: image editing, PDF conversion
  • googlecl: Google Docs support
